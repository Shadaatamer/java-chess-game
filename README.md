# Java Chess Game ♟️

A chess game built in Java using Object-Oriented Programming and a graphical user interface.

## Features

* Full chess board and pieces
* Valid move system
* Turn-based gameplay
* GUI interface

## Technologies

* Java
* Maven
* Swing / NetBeans GUI

## Project Structure

* `ChessGame` – main game logic
* `ChessBoard` – board system
* `Piece` classes – Pawn, King, Queen, etc.
* `ChessGUI` – graphical interface

## How to Run

1. Open in any Java IDE (IntelliJ, NetBeans, Eclipse)
2. Build using Maven (`pom.xml`)
3. Run the GUI/main class
